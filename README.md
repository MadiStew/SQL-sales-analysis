# SQL-Sales-Analysis
Sales data analysis project using SQL (classicmodels database)

This project analyzes a sales database using SQL show highlights about revenue, customer behavior, and product performance.

The dataset used is the classicmodels database.

---
Tools Used

* SQL
* Relational database concepts
* Joins, aggregations, and window functions

---
Questions Answered

1. What is the total revenue?
2. Which products sell the most?
3. What are the sales trends over time?
4. Which customers spend the most?
5. Which product performs best in each category?

---
Insights

* Identified top-performing products by revenue and quantity
* Found seasonal sales trends across months
* Ranked customers by total spending
* Determined best-selling products within each category

---
Skills demonstrated

* Multi-table joins
* Aggregation functions (SUM, GROUP BY)
* Window functions (RANK)
* Data analysis and interpretation
